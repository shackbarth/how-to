Business Object: Terms

Overview

Terms are used to determine the billing Terms for Accounts Payable and Accounts Receivable.

Specifications

  • The Terms type will contain Days
  • The Terms type will contain Proximo
  • Terms Type Days should be allowed to be set.
  • When terms type is Days, Due Days will be a positive number.
  • When terms type is Days, the Cut Off Day is Read-Only.
  • The Terms type Proximo should be allowed to be set.
  • When terms type is Proximo, the Due Date is set to 1.
  • When terms type is Proximo, the Discount Day is set to 1.
  • When terms type is Proximo, the Cut Off Day is editable.
  • When term type is Proximo, the Cut Off Day will allow values between and including 0 and 31.
  • When term type is Proximo, the Due Date will allow values between and including 0 and 31.
  • There will be a drop-down that contains the default Terms (Days & Proximo).
  • Terms uses a function call calculateDueDate().
  • When the Term type is Days, the due date is the Start Date plus the terms Due Days.
  • When the Term type is Proximo, if the start date is less than the terms cutoff date then the due date is the Due Days of the current month.
  • When the Term type is Proximo, if the start date is greater than the terms cutoff date then the due date is the Due Days of the next month.
  • Terms uses a function called calculateDiscountDate().
